The two fonts utilized are mentioned below and can be changed by downloading new ones and changing the global variables below. One can access them in the _variables.scss file.

$primary-font: 'Julius Sans One', sans-serif;

$secondary-font: 'Quicksand', sans-serif;

There is a global variable to create a type scale of the font as well as for the line height. They can be manipulated in the _variables.scss file.

$base-font-size: 1.2rem;

$base-line-height: 1.75rem;

There are three main colors that are used in the framework than can be changed by simply modifying the below variables inside _variables.scss.

$primary-color: #2b3a67;

$secondary-color: #e84855;

$tertiary-color: #fff0f5;
