The responsive navigation bar adapts to two different screen sizes and utilizes the unordered list element to display all of the links deemed necessary.

Each item below has class="list-item" positioned inside a ul tag with a class="flex flex-row space-evenly no-list" in order to display them evenly spaced out, with no list marks and horizontally. All of this is wrapped inside a nav tag with class="nav-bar".

The user can also create a dropdown item so that more options can be added to the navigation seamlessly. A div element with class="dropdown" must be added inside of the ul tag. Then, add a button with class="dropdown-button" with an accompanied div element with class="dropdown-content" which will hold all of the li and a elements necessary.

  • Item 1

The breakpoints added will allow the navigation bar to adapt to two screen sizes, specifically for iPhone X and iPad. It will change the flex direction of the nav element so that it can fit the specified screen.

First media query: iPhone X

min-device-width: 375px;

max-device-width: 812px;

device-pizel-ratio: 3;

First media query: iPad

min-device-width: 834px;

max-device-width: 1112px;

device-pizel-ratio: 2;

The user is capable of changing such dimensions in the _breakpoints.scss file.
